# Component samples on nextjs

Mainly use UI libraries.

## How to launch app.

Install modules.

```bash
npm install
```

Launch Application as development mode.

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
